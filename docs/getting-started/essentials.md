# Essentials

Use this page to document:

- which command-line tools belong in the baseline
- which desktop applications belong in the baseline
- which items are required on every machine versus optional

The current baseline starts with [Xcode](xcode.md) and the Xcode Command Line
Tools so the rest of the developer toolchain can be installed cleanly.

Use `./bootstrap/bootstrap.sh` as the entrypoint for the repo-managed setup
steps that exist at a given point in time.
